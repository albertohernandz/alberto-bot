###Alberto Bot is a sticker bot built in the library of [whatsapp-web.js](https://wwebjs.dev/ "whatsapp-web.js")

## Requirements:
- Install NodeJs
[Download](https://nodejs.org/en "Download")

- Install Git
[Download](https://git-scm.com/ "Download")

## Verification:
- Open a command line or terminal and run the following commands:

- node version
```
node --version
```

- npm version
```
npm --version
```

- git version
```
git --version
```

- You can also use -v after the command to check the version

# ‚≠êINSTALLATION
- Open a command line or terminal and run the following commands:

- Clone repository
```sh
git clone https://github.com/albertohernandz/alberto-bot.git
```

- Enter the folder
```sh
cd alberto-bot
```

- List the files
```sh
ls
```

- Install dependencies
```sh
npm install
```

- Run
```sh
npm run index
```

## Start the connection!

- Bot launch banner when running <b>npm run index</b> command

# ALBERTO BOT REPOSITORY
## Installation
- Clone Repository
```
git clone https://github.com/albertohernandz/alberto-bot.git
```
- Access Folder
```
cd alberto-bot
```
- List Files
```
ls
```
- Install Dependencies
```
npm install
```
- Run
```
npm run index
```
